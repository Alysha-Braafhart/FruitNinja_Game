# Fruit Ninja

## Feedback in lagen
- De speler ziet allemaal fruit op het scherm
- Bij elke fruit dat ze snijden wordt de score hoger
- Het fruit beweegt omhoog het scherm op en als je het open snijd wordt het geanimeerd
- Het fruit krijgt veel kleuren en geluid als je het snijd

## Breakdown mechanic
De mechanic start als de speler met de muis over een stuk fruit gaat
